Drag the slider to simulate your finger moving in front of the webcam. Your calibration defined 0.4 as "Closed" and 3.0 as "Open".

2. Normalization (The Universal Scale)

The code converts any hand size into a standard 0.0 to 1.0 percentage.

(0.40 - 0.4) / (3.0 - 0.4) = 0.00

3. Servo Command (Physical Action)

Finally, we map that percentage to your servo's physical range (20° to 160°).

Target Servo Angle: 20.0°
