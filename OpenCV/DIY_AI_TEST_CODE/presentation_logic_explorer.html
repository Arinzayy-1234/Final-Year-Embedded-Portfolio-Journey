InMoov Logic: The Normalization Pipeline

This interactive tool demonstrates how we convert raw camera data into precise robotic movements.

Stage 1: Raw Camera Ratio (Tip-to-Wrist / Palm) 0.40

Formula: ratio = distance(Tip, Wrist) / distance(Knuckle, Wrist)

Stage 2: Normalization (0% to 100%) 0.00

Formula: normalized = (current - closed) / (open - closed)

Stage 3: Servo Mapping (Physical Degrees) 20.0°

Formula: angle = servo_min + (normalized * (servo_max - servo_min))
